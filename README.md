
# Gradio Chat App with Transmission Policy Specifier

## Features
- GPT-4 assistant
- Mode-aware behavior: private / personalized / sharing
- Transmission Principle Reasoner (advisory roles)
- Transmission Policy Specifier: Set data uses and recipients
- Policy stored per session and attached to each message

## Run
```bash
pip install -r requirements.txt
python chat_app.py
```
